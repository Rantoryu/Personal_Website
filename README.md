## 🚀 Run Locally

Clone the project

```bash
  git clone https://github.com/Rantoryu/Personal_Website.git
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python .\manage.py runserver
```

## 📝 Tech Stack

Python, Django, JavaScript, CSS3, HTML5, Railway, Sqlite3
